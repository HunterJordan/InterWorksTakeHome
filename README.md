### Before running:
Add a .env file to the root of the directory with the following variables

```commandline
DB_HOST=****
DB_USER=****
DB_PASS=****
DB_DATABASE=****
DB_PORT=****
```
### To Run

```bash
docker compose up pyscript --build -d
```

- python
- pandas
- numpy
- sqlalchemy

- docker-compose
